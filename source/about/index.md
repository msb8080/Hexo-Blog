## 🚀 项目经历

### OmniAgent 多智能体平台
- 基于 LangGraph 的多 Agent 编排框架
- 支持 RAG、工具调用、记忆管理
- 前后端分离架构，支持私有化部署

### 企业知识库系统
- 基于 RAG 的智能问答系统

- 支持角色一致性控制
